# 【开源工具】闲鱼自动化搜索工具 - 支持关键词、价格筛选、个人闲置

## 工具简介

大家好！今天分享一个我开发的闲鱼自动化搜索工具，基于Selenium实现，功能完整稳定，特别适合需要批量搜索闲鱼商品的用户。

**项目地址**: https://github.com/yourusername/xianyu-search

## 开发背景

在使用闲鱼过程中，经常需要：
- 搜索特定商品
- 设置价格范围
- 筛选个人卖家
- 批量获取商品信息

手动操作效率低下，于是开发了这个自动化工具。

## 核心功能

### 1. 智能搜索 🔍
- 支持任意关键词搜索
- 自动点击"最新发布"排序
- 保持浏览器窗口显示顺序

### 2. 精确筛选 💰
- 价格范围筛选（避免400变4000的bug）
- 个人闲置自动勾选
- 地区信息提取

### 3. 完整数据 ⏰
- 支持所有时间格式：分钟前、小时前、天前、天内、刚刚、今天、昨天、前天
- 避免商品描述被错误识别为时间

### 4. 实用输出 🔗
- 生成可点击的web_link和deep_link
- 飞书兼容：deep_link可直接点击
- 自动保存JSON数据和截图

## 技术特点

### 优化亮点
- ✅ **排序正确**：保持浏览器窗口显示顺序
- ✅ **时间抓取完整**：支持所有时间格式
- ✅ **错误处理完善**：避免商品描述被错误识别
- ✅ **兼容性好**：支持macOS、Linux、Windows

### 技术栈
- Python 3.7+
- Selenium 4.0+
- Requests + BeautifulSoup4
- Chrome + ChromeDriver

## 快速开始

### 安装依赖
```bash
pip install selenium requests beautifulsoup4 lxml
```

### 基本使用
```bash
# 搜索野豹球杆，价格1000-3000
python3 xianyu_clickable_links.py "闲鱼搜" "野豹球杆" 1000 3000

# 搜索个人闲置的奥秘梦幻，价格200-600
python3 xianyu_clickable_links.py "闲鱼搜" "奥秘梦幻" 200 600 "个人"
```

### OpenClaw集成
```bash
# 作为OpenClaw技能使用
闲鱼搜 关键词 最低价 最高价 [个人]
```
