# 闲鱼搜索技能 - 发布说明

## 技能简介
基于Selenium的闲鱼（goofish.com）自动化搜索技能，经过多轮优化，功能完整稳定。

## 主要功能
1. **关键词搜索**：支持任意关键词搜索
2. **价格筛选**：精确价格范围筛选
3. **个人闲置**：自动勾选个人闲置筛选
4. **智能排序**：保持浏览器窗口显示顺序
5. **时间抓取**：完整支持所有时间格式
6. **链接生成**：生成可点击的web_link和deep_link

## 优化亮点
- ✅ **排序优化**：保持浏览器窗口显示顺序（最新发布在前）
- ✅ **时间抓取**：支持分钟前、小时前、天前、天内、刚刚、今天、昨天、前天等所有格式
- ✅ **错误修复**：避免商品描述被错误识别为时间
- ✅ **兼容性**：飞书deep_link可直接点击

## 版本历史
- v1.0.0: 初始版本，规范链接格式
- v1.0.1: 修复排序问题，保持浏览器顺序
- v1.0.2: 优化时间抓取，扩展匹配模式
- v1.0.3: 进一步优化时间抓取，添加多种提取方法
- v1.0.4: 修复时间抓取错误，添加长度限制
- v1.0.5: 新增x天内、前天格式支持（当前版本）

## 使用要求
1. **Python 3.7+**
2. **Chrome浏览器**
3. **ChromeDriver**（与Chrome版本匹配）
4. **Python依赖**：selenium, requests, beautifulsoup4, lxml

## 安装命令
```bash
clawhub install xianyu-search
```

## 使用示例
```bash
# 基本搜索
闲鱼搜 野豹球杆 1000 3000

# 个人闲置搜索
闲鱼搜 奥秘梦幻 200 600 个人

闲鱼搜 测试 100 200
```
