🎯 闲鱼搜索工具

基于Selenium的闲鱼自动化搜索工具 - 功能强大，使用简单

✨ 功能特性

🔍

智能搜索

支持任意关键词搜索，快速找到目标商品

💰

价格筛选

精确价格范围筛选，避免无效结果

👤

个人闲置

自动筛选个人卖家，交易更放心

⏰

时间抓取

完整支持所有时间格式，信息更全面

🚀 快速开始

安装依赖

pip install selenium requests beautifulsoup4 lxml

基本使用

# 基本搜索
python3 xianyu_clickable_links.py "闲鱼搜" "野豹球杆" 1000 3000

# 个人闲置搜索
python3 xianyu_clickable_links.py "闲鱼搜" "奥秘梦幻" 200 600 "个人"

OpenClaw集成

# 作为OpenClaw技能使用
闲鱼搜 关键词 最低价 最高价 [个人]
