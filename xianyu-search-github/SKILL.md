## 📋 技能描述

基于Selenium的闲鱼（goofish.com）自动化搜索技能，支持关键词搜索、价格筛选、个人闲置过滤，并生成可点击的商品链接。

## 🚀 快速开始

### 基本命令格式
```
闲鱼搜 [关键词] [最低价] [最高价] [个人]
```

### 使用示例
1. **基本搜索**: `闲鱼搜 野豹球杆 1000 3000`
2. **个人闲置**: `闲鱼搜 野豹球杆 1000 3000 个人`
3. **价格筛选**: `闲鱼搜 奥秘球杆 100 500`
4. **简单搜索**: `闲鱼搜 测试 100 200`

## 🔧 技术实现

### 依赖要求
```bash
# Python依赖
pip install selenium requests beautifulsoup4 lxml

# 系统依赖
- Chrome浏览器
- ChromeDriver（与Chrome版本匹配）
```

### 登录机制
1. **首次使用**: 需要扫码登录闲鱼
2. **cookies保存**: 登录信息自动保存到 `cookies/` 目录
3. **后续使用**: 自动加载cookies，无需重复登录

### 搜索流程
```
1. 启动浏览器（Chrome）
2. 加载cookies登录状态
3. 访问闲鱼搜索页面
4. 点击"新发布"按钮
5. 点击"最新"按钮（按发布时间排序）
6. 输入价格范围（最低价-最高价）
7. 悬停显示确认按钮并点击"确定"
8. 抓取商品信息（保持页面原始顺序）
9. 提取商品ID、价格、时间、地区
   - **时间抓取优化**：使用多种方法（正则匹配、关键词匹配、CSS选择器、遍历子元素）
   - **长度限制**：时间文本通常<30字符，避免商品描述被错误识别
   - **支持格式**：分钟前、小时前、天前、天内、刚刚、今天、昨天、前天、日期发布、时间发布
10. 生成链接格式
11. 保存截图和数据
12. 格式化输出结果

排序规则：保持浏览器窗口显示顺序（最新发布在前），不重新排序
```

## 📊 输出格式

### 链接格式
每个商品提供两种链接：

1. **web_link**: `https://www.goofish.com/item?id={product_id}&categoryId=126866353`
   - 网页版链接，在浏览器中打开商品页面

2. **deep_link**: `https://h5.m.goofish.com/item?id={product_id}`
   - 移动网页版链接，自动唤醒闲鱼App
   - **飞书兼容**：在飞书中可直接点击

### 输出内容
- 前10个商品信息（标题、价格、发布时间、地区）
- 商品ID
- web_link 和 deep_link
- 价格分析（最低、最高、平均价格）
- 地区分布统计
- 链接使用说明

### 数据保存
每次搜索自动生成：
- **截图文件**: `clickable_{keyword}_min{min_price}_max{max_price}_{timestamp}.png`
- **数据文件**: `clickable_{keyword}_min{min_price}_max{max_price}_{timestamp}.json`

保存位置: `search-results/` 目录

## 📝 使用建议

### 最佳实践
1. **关键词选择**: 使用具体的关键词获得更准确的结果
2. **价格范围**: 设置合理的价格范围避免结果过多或过少
3. **个人闲置**: 如需个人卖家商品，添加"个人"参数
4. **链接使用**: 在飞书中使用 `deep_link` 可直接点击

### 故障排除
1. **登录失败**: 删除 `cookies/` 目录重新登录
2. **浏览器问题**: 检查Chrome和ChromeDriver版本匹配
3. **无结果**: 调整关键词或价格范围
4. **链接不可点击**: 确保使用 `deep_link` 格式
5. **格式不正确（关键问题）**:
   - **问题现象**: 显示简洁格式（商品 - 价格 (地区，时间)），缺少链接
   - **问题原因**: 误用了 `xianyu_2138_format.py` 而不是 `xianyu_clickable_links.py`
   - **解决方案**: 
     1. 确认使用 `xianyu_clickable_links.py` 作为主程序
     2. 检查命令是否正确：`python3 xianyu_clickable_links.py "闲鱼搜" "关键词" 最低价 最高价`
     3. 确保输出包含 `web_link` 和 `deep_link`
   - **验证方法**: 检查输出是否包含"商品ID"、"web_link"、"deep_link"字段
